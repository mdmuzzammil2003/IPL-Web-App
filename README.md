# 🏏 IPL Analytics Web Application

## Overview
A full stack web application that analyses IPL match 
data and displays team statistics interactively.

Select any IPL franchise and instantly see:
- Total matches played
- Total wins and losses
- Win percentage
- Toss win analysis
- Season by season performance chart
- Top MVP players chart
- Toss decision pie chart

## Tech Stack
- **Backend:** Python, Flask
- **Database:** SQL (SQLite)
- **Data Analysis:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn
- **Frontend:** HTML, CSS

## How to Run
1. Clone this repository
2. Install requirements:
   pip install flask pandas matplotlib seaborn
3. Run the app:
   python app.py
4. Open browser → localhost:5000
